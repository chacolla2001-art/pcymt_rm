## 2. Información que Recopilamos

### 2.1 Información Personal

Cuando utilizas PCyMT RM, podemos recopilar la siguiente información:

#### **A) Mediante Google Sign-In:**
- Nombre completo
- Dirección de correo electrónico
- Foto de perfil (opcional)
- ID de cuenta de Google

#### **B) Mediante Registro Directo:**
- Nombre de usuario
- Dirección de correo electrónico
- Contraseña (almacenada de forma encriptada)
- Nombre y apellido (opcional)

### 2.2 Información de Ubicación

La aplicación solicita acceso a tu ubicación para:
- **Experiencias de Realidad Aumentada:** Mostrar modelos 3D de animales en ubicaciones específicas del Parque de las Culturas y la Madre Tierra en La Paz, Bolivia
- **Funcionalidades del mapa:** Visualizar tu posición en relación con puntos de interés (anchor points)

**Tipos de ubicación recopilada:**
- Ubicación precisa (GPS)
- Ubicación aproximada (red móvil/WiFi)

**Control del usuario:** Puedes denegar el permiso de ubicación, pero esto limitará las funcionalidades de AR y mapa.

### 2.3 Información de Uso de la Aplicación

Recopilamos automáticamente:
- **Interacciones con modelos 3D:** Qué animales visualizas, por cuánto tiempo, rotaciones, zoom
- **Sesiones de usuario:** Fecha y hora de inicio/cierre de sesión, duración
- **Información del dispositivo:** Modelo, versión de Android, resolución de pantalla
- **Datos de cámara:** Solo durante el uso de AR (no se almacenan fotos/videos sin tu consentimiento)

### 2.4 Información que NO Recopilamos

- **Fotos o videos de tu cámara:** La cámara se usa solo para renderizar AR en tiempo real
- **Contactos telefónicos**
- **Archivos personales del dispositivo**
- **Mensajes SMS o llamadas**

## 5. Seguridad de los Datos

### 5.1 Medidas de Protección

Implementamos múltiples capas de seguridad:

- **Encriptación de datos en tránsito:** HTTPS/TLS para todas las comunicaciones
- **Encriptación de datos en reposo:** EncryptedSharedPreferences para almacenamiento local
- **Autenticación segura:** Tokens JWT con expiración automática
- **Contraseñas hasheadas:** bcrypt con salt para contraseñas locales
- **Validación de entrada:** Sanitización de todos los datos de usuario

### 5.2 Almacenamiento de Datos

- **Datos locales:** Almacenados de forma encriptada en el dispositivo (preferencias de usuario, tokens de sesión)
- **Datos remotos:** Almacenados en base de datos PostgreSQL con backups regulares
- **Caché:** Datos temporales eliminados al cerrar sesión

### 5.3 Retención de Datos

- **Datos de cuenta:** Se mantienen mientras tu cuenta esté activa
- **Datos de sesión:** Se eliminan después de 7 días de inactividad
- **Datos de interacción:** Se conservan de forma anónima para análisis estadístico
