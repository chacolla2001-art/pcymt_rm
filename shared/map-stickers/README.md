# Stickers del mapa del parque

Fuente única de los 44 stickers SVG/PNG usados en el mapa canvas.

**Consumidores:**
- `apps/web-admin` — copiados en build vía `angular.json`
- `apps/mobile-android` — sincronizados en build vía tarea Gradle `syncMapStickers`

Edita los archivos aquí; no modifiques copias locales en las apps.
